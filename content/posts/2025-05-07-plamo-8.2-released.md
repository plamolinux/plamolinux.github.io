Plamo 8.2 をリリースしました。

<!--more-->

```
たいてい年に一度は公式版を出してたものの，昨年はあれこれ忙しくて時期を逸っしてしまい，
ずるずると日付け入りのテスト版でしのいできましたが，さすがにそろそろケリを
付けた方がいいかと思い，正式版をリリースすることにしました．

もっとも「正式版」と言っても，各パッケージは随時更新しているので，
従来からお使いの方はそのまま get_pkginfo 等でパッケージを更新し続けられます．
「正式版」の意味は，年に一度くらい iso イメージを更新しておかないと，
それを使って新規インストールした後のパッケージ更新が多量になるから，程度の意味です．

主要なパッケージのバージョンは以下の通りです．

kernel  : linux-6.12.21(Nvidia ドライバは 570.133.07)
GCC     : 14.2.0
LLVM    : 19.1.0
Mesa3d  : 25.0.3
Xfce    : 4.20
LXqt    : 2.1
Mate    : 1.28
FireFox : 137.0
ffmpeg  : 7.1.1
texlive : 20240313
LibreOffice : 24.8.6

容量増加の結果，DVD用のisoイメージは3枚組みで以下のような構成になっています．

・1枚目(dvd_01.iso)
  インストーラ，plamo/00_base ～ 07_multimedia, 09_printings,l 10_xfce, 11_lxat
・2枚目(dvd_02.iso)
   plamo/08_daemons, 12_mate, 14_libreoffice, 15_kernelsrc, 16_virtualization
・3枚目(dvd_03.iso)
   plamo/13_tex

USBイメージ(usb.iso)も12GBくらいになっているのでご注意ください．

今回，新な試みとして，インストール時には不要なネットワーク関連のツールや firmware を削除し，
インストーラをシェイプアップ(軽量化)してみました．
そのため，以前は対応していたハードウェアが(firmwareの不足で)使えなくなっている
可能性があります．そのような状況に遭遇されましたら，ご報告いただければ助かります．

合わせて今回から，VFAT形式のUSBメモリから起動するためのツール(Plamo8_USBboot_img.zip)を
作ることにしました．このzipファイルに含まれているEFI, boot, grub の3ディレクトリを
USBメモリにコピーすれば，UEFI経由でUSBメモリから起動できます(UEFIの実装にも依存しますが)．
こちらの環境にはネットワーク用のツールやlinux-firmware一式を含めているので，
レスキューシステムとして使ったり，上記インストーラでは(firmwareが無くて)使えない
ハードウェアにも対応できるかも知れません．
この環境にもインストール用のスクリプト一式は揃えているので，
USBメモリ上にplamo/以下のパッケージを置いたり，別途用意したDVDからパッケージを
インストールすることもできます．
```
