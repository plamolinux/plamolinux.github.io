ntp 4.2.8p10

 2017-04-04に投稿  |  Plamo Linux Maintainer Group

ntp にセキュリティホールが発見されています。詳しくは以下をご覧ください。

  • March 2017 ntp-4.2.8p10 NTP Security Vulnerability Announcement

対策パッケージをリリースしました。

  • ntp-4.2.8p10-i686-P1.txz (32bit)
  • ntp-4.2.8p10-x86_64-P1.txz (64bit)

ntp-4.2.8p9-*-P2 パッケージより、動作に ntp ユーザ・グループが必要になりました。詳しくは Change.Log をご覧ください。
